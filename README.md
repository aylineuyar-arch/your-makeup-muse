# Aura — A Loveable App

A loveable application that maps your skin tone, undertone, face shape, and eye shape to a curated kit and a step-by-step makeup routine tailored to your skill level.

## About

Aura helps you move beyond generic trends by composing a singular beauty signature based on your unique features:

- **Signature Kit** — base, eye, cheek, and lip products picked for your tone and undertone
- **Application Protocol** — a step-by-step routine that scales from beginner to expert
- **Brush Library** — the right tool, in the right order
- **Shape Guides** — interactive helpers if you're not sure of your face or eye shape

## Built with Lovable

This project was built with [Lovable](https://lovable.dev) — an AI-powered app builder.

Tech stack:
- TanStack Start (React 19 + Vite 7)
- Tailwind CSS v4
- TypeScript

## Development

```bash
bun run dev
```
